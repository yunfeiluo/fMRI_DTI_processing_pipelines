# fMRI_DTI_processing_pipelines 
fMRI data processing pipelines  

# Components 
## fMRI pre-processing pipeline 
- skull strip  
- motion correction  
- registration  
- segmentation  
- normalization  
## DTI processing pipeline 
- normalization
- fiber tracking

- afni 
- DIPY
